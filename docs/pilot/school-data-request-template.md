# School Data Request Template

This document is designed to be sent to each pilot school before setup begins.

## Subject

EduMyles Pilot Setup Requirements for [School Name]

## Message

Dear [School Name],

Thank you for agreeing to participate in the EduMyles pilot.

To prepare your school workspace, we need the following information before setup:

## 1. School Profile

- official school name
- school type
- county and location
- official school phone number
- official school email address
- school logo
- website or social media page if available

## 2. Main Contacts

- pilot champion name
- pilot champion phone number
- pilot champion email address
- backup contact name
- backup contact phone number
- backup contact email address

## 3. Staff List

Please share a sheet with:

- first name
- last name
- email
- phone number
- role
- department

Suggested roles:

- school admin
- principal
- teacher
- bursar
- hr manager
- librarian
- transport manager

## 4. Student List

Please share a sheet with:

- first name
- last name
- date of birth
- gender
- class
- admission number
- guardian name
- guardian email
- guardian phone
- guardian relationship

## 5. Class Structure

- levels or grades
- streams or sections
- class teachers if available

## 6. Finance Setup

- current fee structure
- term name
- academic year
- class-based fee differences if any
- preferred payment methods for pilot

## 7. Parent Access

Please confirm whether parents should be onboarded during phase 1 of the pilot.

If yes, please ensure guardian contact details are complete and accurate.

## 8. Pilot Start Date

Please confirm:

- preferred kickoff date
- preferred training date
- preferred go-live date

## 9. File Format

Please share data in Excel or CSV format where possible.

## 10. Important Note

To avoid onboarding delays, please make sure:

- names are correctly spelled
- phone numbers are valid
- email addresses are active
- class names are consistent across all files

Thank you,

EduMyles Team
