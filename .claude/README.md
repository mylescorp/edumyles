# Claude Workspace

This directory contains workspace-specific configurations and files for the Claude AI assistant integration with the EduMyles project.

## Purpose

- Stores Claude-specific workspace settings
- Maintains context for AI-assisted development
- Preserves worktree configurations for parallel development

## Structure

- `worktrees/` - Parallel development worktrees
- Configuration files for Claude integration
- Context preservation files

## Usage

This directory is managed automatically by the Claude IDE and should not be manually modified unless specifically required for workspace configuration changes.
