# EduMyles Launch Video

Remotion source for a 45-second EduMyles launch video.

## Storyline

- EduMyles as one platform for school operations.
- Students, fees, timetables, exams, and staff in one dashboard.
- Multi-school subdomain architecture for isolated school dashboards.
- M-Pesa fee payments and reconciliation.
- Parent, teacher, and admin experiences.
- Closing EduMyles logo with `edumyles.com`.

## Commands

```bash
npm install
npm run studio
npm run render
```

The rendered MP4 is written to `out/edumyles-launch.mp4`.
